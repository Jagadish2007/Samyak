DATABASE TABLES (Auto-created)
-------------------------------
1. users
   - id (Primary Key, Auto Increment)
   - name
   - email (Unique)
   - password
   - role (ADMIN/STUDENT)

2. courses
   - id (Primary Key, Auto Increment)
   - title
   - description

3. enrollments
   - id (Primary Key, Auto Increment)
   - user_id (Foreign Key to users)
   - course_id (Foreign Key to courses)


API ENDPOINTS
-------------
BASE URL: http://localhost:8080/api

Authentication:
  POST   /auth/register          Register new user
  POST   /auth/login             Login user

Courses:
  GET    /courses                Get all courses
  POST   /courses                Create new course

Enrollments:
  POST   /enroll                 Enroll user in course
  GET    /enrollments/{userId}   Get user enrollments

HOW TO RUN
----------
1. Install Java 17+, Maven, MySQL
2. Create database: CREATE DATABASE lms_db;
3. Update password in: src/main/resources/application.properties
4. Run: mvn spring-boot:run
5. Access: http://localhost:8080

TROUBLESHOOTING
---------------
Issue: Can't connect to database
Fix: Check MySQL is running and password is correct

Issue: Port 8080 in use
Fix: Change server.port in application.properties

Issue: Tables not created
Fix: Ensure spring.jpa.hibernate.ddl-auto=update

Issue: Maven command not found
Fix: Add Maven to system PATH and restart terminal
